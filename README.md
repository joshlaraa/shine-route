ShineRoute is a full-stack web application for mobile auto detailers. It helps manage daily appointments and automatically creates the most efficient route for the day.

## Features

- Add and manage appointments
- Address autocomplete
- Map view of all stops
- Smart route optimization (minimizes drive time)
- View total distance and ETA
- Mobile-friendly interface for on-site use

## Tech Stack

- Frontend: Next.js + Tailwind
- Backend: Next.js API Routes / Node
- Database: Supabase
- Mapping: Google Maps API (Geocoding, Places, Directions)

## Getting Started

Install dependencies:

```
npm install
```

Run the dev server:

```
npm run dev
```

Environment variables (`.env.local`):

````
GOOGLE_MAPS_API_KEY=your_key
SUPABASE_URL=your_url
SUPABASE_KEY=your_key
```x
````
